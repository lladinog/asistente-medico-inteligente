- diagnostico: Cuando el usuario describe síntomas o busca un diagnóstico
- analisis_imagenes: Cuando el usuario menciona imágenes médicas, radiografías, etc.
- interpretacion_examenes: Cuando el usuario pregunta sobre resultados de exámenes de laboratorio
- explicacion: Cuando el usuario pide explicaciones sobre condiciones médicas o términos
- buscador_centros: Cuando el usuario busca centros médicos o ubicaciones
- contacto_medico: Cuando el usuario quiere contactar o simular contacto con un médico

Responde ÚNICAMENTE con el nombre de la funcionalidad correspondiente, sin explicaciones ni texto adicional.
